Determine if a ray hits the object.

This method checks if a given ray intersects with the object within a specified range. If an intersection occurs, it populates the hit record with details about the intersection.

Parameters
rThe ray to check for intersection.
t_minThe minimum value of the parameter t to consider.
t_maxThe maximum value of the parameter t to consider.
recThe hit record to populate with intersection details.
Returns
True if the ray intersects the object, false otherwise.
