PotatRaycer
A simple raytracer in c++
Configure a scene file

The scene file is a .cfg file, used to execute the program after his compilation.\ All the values inside this files must be float and respect the same syntax of the following exemples.\ It contains the following configuration :

logger configuration :

logger : {
state : "false" // or, true
}

Camera configuration :

camera : {
lookfrom : { x = 0.0; y = 0.0; z = 0.0; },
lookat : { x = 0.0; y = 0.0; z = 0.0; },
vup : { x = 0.0; y = 0.0; z = 0.0; },
vfov : { x = 0.0; },
width : { x = 0.0; },
height : { x = 0.0; },
aperture : { x = 0.0; },
ns : { x = 0.0; },
}

Light configuration :

light : {
type : "directional", // or, ambiant
position : { x = 0.0; y = 0.0; z = 0.0; },
intensity : { x = 0.0; },
normal : { x = 0.0; y = 0.0; z = 0.0; }, // only for directional light
size : { x = 0.0; } // only for directional light
}
